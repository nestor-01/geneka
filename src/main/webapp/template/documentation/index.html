Help & Tips

Do not start from scratch, use an existing page we have created and modify it to learn how it works.

Getting an error message? Chances are someone else has seen it to, try a Google search for a quick fix.

Many times plugins are to blame when a site fails, always try disabling any third-party plugins if you encounter an error you can't solve.

Don't forget to re-explore the live demo for layouts, usage ideas and sample code.

NOTE: The whole Theme is build with Twitter Bootstrap Framework. For a better understanding visit their page and get to know all the css classes.

HTML Structure

The "Chappi" Theme is based on Twitter Bootstrap which looks beautiful at any size, be it a 17" laptop screen or an iPhone.

It has the flexibility to go mobile like a champ. You can easily customize all the parts of site or build new.

IMPORTANT: You have to use 2 data-attributes.
1. data-scroll-nav="x" wich defines your Menu.
2. data-scroll-index="x" which defines your section.
Please take a look at our Code before you change.

You can set up your own options in "custom.js"

// One Page Navigation

          easing: 'easeInOutExpo',
          topOffset: -20,
        });

// Phone Carousel            
      var mySwiper = $('.swiper-container').swiper({
      
         mode:'horizontal',
         loop: true,
         speed: 950,
         slidesPerView: getSlide(),
         grabCursor: true,
         roundLengths: true,
         tdFlow: {
              rotate: -30,
              stretch: 10,
              depth: 120,
              modifier: 1,
              shadows: false
          }
         
      });    
      
      // Set number of slide based on device width
      $(window).resize(function() {
           var wW = $(window).width();
           if (wW < 601) {
               mySwiper.params.slidesPerView = 1;
           } else {
               mySwiper.params.slidesPerView = 3;
           }
           mySwiper.reInit();
      });
      
      // Arrow Click Phone Carousel
      $('.arrow-left').on('click', function(e){
         e.preventDefault()
         mySwiper.swipePrev()
      })
      
      $('.arrow-right').on('click', function(e){
         e.preventDefault()
         mySwiper.swipeNext()
      })

Parallax Backgrounds

If you want the "parallax" version, take a look at index3.html in your download folder. The set up is very simple.

First of all, add the body class ".parallax" in your HTML Code
<body class="parallax">

Then go to your section, in this case we take the "Header":

<header style="background-image: url(img/bg/big-images/1.jpg)" data-stellar-background-ratio="0.3">

as you can see, we need a special data-attribute:

data-stellar-background-ratio="0.3"

Add this line to your sections and you are done. The parallax effect is ready for your images.

NOTE: Please visit https://github.com/markdalgleish/stellar.js for more detailed information about the options you have.

Scroll Effects

You have the option to make your own great scrolling effects. Only a few things are important.

The structure looks as follows:

<div class="swiper-container wow fadeIn" data-wow-delay="1.5s" data-wow-delay="0.4s">

First of all you need 2 important classes for your item to be animated: "wow" and "fadeInDown"

In this case "fadeInDown" is the animated class. Of course you can choose whatever you want, just visit this page: http://daneden.github.io/animate.css/ and take your own personal animated class.

The second part of the animation are the special data-attributes from the plugin.

In this case:

data-wow-delay="0.4s"
For more information please visiti this website: http://mynameismatthieu.com/WOW/docs.html
